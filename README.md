# Sales Performance Dashboard (Power BI)

## Overview
This project analyzes sales performance using Power BI to identify trends,
top-performing regions, products, and customer segments.

## Tools Used
- Power BI
- Excel

## Key Insights
- Sales show seasonal trends with peak periods
- Certain regions outperform others in revenue and profit
- Some high-sales products have low profit margins

## Recommendations
- Focus on high-performing regions
- Review pricing for low-margin products
- Plan inventory around seasonal demand
